## Wie zu verifizieren

### Auf BSCScan

1. Besuchen Sie [bscscan.com](https://bscscan.com)
2. Kopieren Sie die Vertragsadresse in die Suchleiste
3. Klicken Sie auf den "Contract" Tab
4. Überprüfen Sie, dass ein ✅ "Verified" Badge angezeigt wird

### In Ihrer Wallet prüfen

1. Öffnen Sie Ihren Transaktionsverlauf
2. Klicken Sie auf eine IMP-Transaktion
3. Überprüfen Sie, dass die "To" Adresse übereinstimmt
