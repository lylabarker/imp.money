# Erste Einzahlung

Dieser Leitfaden führt Sie durch Ihre erste USDT-Einzahlung auf IMP Money.

## Voraussetzungen

Bevor Sie einzahlen:
- ✅ Wallet verbunden (MetaMask oder andere)
- ✅ Auf BNB Smart Chain-Netzwerk
- ✅ USDT (BEP-20) in Ihrer Wallet
- ✅ Etwas BNB für Gasgebühren (~$0,30)

## USDT für Einzahlung besorgen

### Option 1: Von Börse senden
1. Kaufen Sie USDT auf Binance, Coinbase usw.
2. Wählen Sie **BEP-20 (BSC)** Netzwerk beim Abheben
3. Senden Sie an Ihre Wallet-Adresse
4. Warten Sie auf Bestätigung (~3 Minuten)

### Option 2: Im Wallet tauschen
1. Wenn Sie BNB oder andere Token haben
2. Verwenden Sie den integrierten Swap von MetaMask
3. Oder verwenden Sie PancakeSwap direkt
4. Tauschen Sie zu USDT

## Schritt-für-Schritt Einzahlung

### Schritt 1: Gehen Sie zur Einzahlungsseite
Klicken Sie auf "Einzahlen" im Hauptmenü von imp.money

### Schritt 2: Betrag eingeben
- Minimum: $10 USDT
- Maximum: Unbegrenzt
- Empfohlen: Starten Sie mit einem Betrag, der für Sie angenehm ist

### Schritt 3: Referral-Link (Falls vorhanden)
Falls jemand Sie eingeladen hat, ist deren Referral-Link bereits angewendet.

### Schritt 4: USDT-Nutzung genehmigen
1. Klicken Sie auf "Genehmigen"
2. MetaMask-Popup erscheint
3. Bestätigen Sie die Transaktion
4. Warten Sie auf Bestätigung

### Schritt 5: Einzahlung bestätigen
1. Klicken Sie auf "Einzahlen"
2. MetaMask-Popup für Einzahlung
3. Überprüfen Sie den Betrag und Gas
4. Bestätigen Sie die Transaktion

### Schritt 6: Warten auf Bestätigung
- Transaktion dauert normalerweise 3-15 Sekunden
- Sie sehen eine Bestätigungsmeldung
- Ihr Dashboard wird aktualisiert

## Nach der Einzahlung

### Was passiert jetzt?
- ✅ Ihre Einzahlung wird im Dashboard angezeigt
- ✅ 0,7% ROI beginnt sofort zu akkumulieren
- ✅ 21-Tage-Zyklus startet
- ✅ Sie können jederzeit Gewinne abholen

### Wichtige Daten
- **ROI Start**: Sofort
- **Erste Abhebung**: Nach 24 Stunden verfügbar
- **Kapital freigeschaltet**: Nach 21 Tagen

## Häufige Fragen

**Warum zwei Transaktionen?**
Die erste genehmigt den Smart Contract, USDT zu verwenden. Die zweite führt die eigentliche Einzahlung durch.

**Was wenn die Transaktion fehlschlägt?**
Überprüfen Sie, ob Sie genug BNB für Gas haben. Versuchen Sie es erneut mit höherem Gas-Limit.

**Kann ich mehr einzahlen?**
Ja! Jede Einzahlung startet ihren eigenen 21-Tage-Zyklus.
