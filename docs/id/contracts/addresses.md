## Cara Memverifikasi

### Di BSCScan

1. Kunjungi [bscscan.com](https://bscscan.com)
2. Tempel alamat kontrak di kolom pencarian
3. Klik tab "Contract"
4. Konfirmasi ada badge ✅ "Verified"

### Periksa di Dompet Anda

1. Buka riwayat transaksi Anda
2. Klik transaksi IMP
3. Verifikasi alamat "To" cocok
