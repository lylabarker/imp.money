## Mengatur MetaMask

### Langkah 1: Install MetaMask
1. Kunjungi [metamask.io](https://metamask.io)
2. Klik "Download"
3. Pilih browser atau perangkat mobile Anda
4. Install ekstensi

### Langkah 2: Buat Dompet
1. Klik "Buat dompet baru"
2. Buat password yang aman
3. **Tulis 12 kata seed phrase Anda**
4. JANGAN simpan secara digital
5. Konfirmasi seed phrase Anda

### Langkah 3: Tambahkan BNB Smart Chain
1. Klik dropdown jaringan (atas)
2. Klik "Tambah jaringan"
3. Masukkan:
   - Nama Jaringan: `BNB Smart Chain`
   - RPC URL: `https://bsc-dataseed.binance.org/`
   - Chain ID: `56`
   - Simbol: `BNB`
   - Explorer: `https://bscscan.com`
4. Simpan

## Menghubungkan ke IMP Money

### Langkah 1: Kunjungi Aplikasi
Pergi ke [imp.money](https://imp.money)

### Langkah 2: Klik "Hubungkan Dompet"
Cari tombol di pojok kanan atas

### Langkah 3: Pilih Dompet Anda
Pilih MetaMask (atau dompet pilihan Anda)

### Langkah 4: Setujui Koneksi
1. Popup MetaMask muncul
2. Verifikasi website (imp.money)
3. Klik "Hubungkan"
4. Pastikan Anda di BNB Smart Chain

### Langkah 5: Verifikasi
Alamat dompet Anda seharusnya muncul di pojok kanan atas
