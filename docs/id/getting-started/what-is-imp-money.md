# Apa itu IMP Money?

IMP Money (Internet Money Protocol) adalah protokol DeFi terdesentralisasi di BNB Smart Chain yang menghasilkan hasil melalui penyediaan likuiditas otomatis di PancakeSwap V3.

## Masalahnya

Yield farming DeFi tradisional itu kompleks:
- Membutuhkan pemahaman tentang liquidity pools
- Manajemen posisi manual
- Biaya gas tinggi untuk rebalancing
- Risiko impermanent loss
- Hambatan teknis untuk pengguna biasa

## Solusinya

IMP Money menyederhanakan semuanya:
- **Deposit satu klik** - Cukup deposit USDT
- **Manajemen otomatis** - Protokol menangani likuiditas
- **Fokus stablecoin** - Impermanent loss minimal
- **Return harian** - Dapatkan 0,7% harian dari deposit Anda
- **Reward jaringan** - Bangun tim untuk melipatgandakan penghasilan
