# Sistem Referral 21 Tingkat

IMP Money memiliki salah satu program referral paling dermawan di DeFi - struktur komisi 21 tingkat yang membayar **total 14%** komisi dari semua deposit.

## Cara Kerjanya

Ketika seseorang deposit menggunakan link referral Anda:

1. **Anda mendapat komisi** (dibayar instan dalam USDT)
2. **Mereka bergabung ke jaringan Anda** di level 1
3. **Referral mereka menjadi level 2 Anda** dan seterusnya
4. **Anda mendapat dari 21 level** kedalaman jaringan
