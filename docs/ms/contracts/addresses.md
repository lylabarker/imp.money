## Cara Mengesahkan

### Di BSCScan

1. Lawati [bscscan.com](https://bscscan.com)
2. Tampal alamat kontrak dalam bar carian
3. Klik tab "Contract"
4. Sahkan ada badge ✅ "Verified"

### Semak di Dompet Anda

1. Buka sejarah transaksi anda
2. Klik transaksi IMP
3. Sahkan alamat "To" sepadan
