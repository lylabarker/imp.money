## Menyediakan MetaMask

### Langkah 1: Pasang MetaMask
1. Lawati [metamask.io](https://metamask.io)
2. Klik "Muat Turun"
3. Pilih pelayar atau peranti mudah alih anda
4. Pasang sambungan

### Langkah 2: Buat Dompet
1. Klik "Buat dompet baharu"
2. Buat kata laluan yang selamat
3. **Tulis 12 perkataan seed phrase anda**
4. JANGAN simpan secara digital
5. Sahkan seed phrase anda

### Langkah 3: Tambah BNB Smart Chain
1. Klik dropdown rangkaian (atas)
2. Klik "Tambah rangkaian"
3. Masukkan:
   - Nama Rangkaian: `BNB Smart Chain`
   - RPC URL: `https://bsc-dataseed.binance.org/`
   - Chain ID: `56`
   - Simbol: `BNB`
   - Explorer: `https://bscscan.com`
4. Simpan

## Menyambung ke IMP Money

### Langkah 1: Lawati Aplikasi
Pergi ke [imp.money](https://imp.money)

### Langkah 2: Klik "Sambung Dompet"
Cari butang di penjuru kanan atas

### Langkah 3: Pilih Dompet Anda
Pilih MetaMask (atau dompet pilihan anda)

### Langkah 4: Luluskan Sambungan
1. Popup MetaMask muncul
2. Sahkan laman web (imp.money)
3. Klik "Sambung"
4. Pastikan anda di BNB Smart Chain

### Langkah 5: Sahkan
Alamat dompet anda patut muncul di penjuru kanan atas
