# Primeiro Depósito

Este guia irá guiá-lo através do seu primeiro depósito de USDT no IMP Money.

## Pré-requisitos

Antes de depositar:
- ✅ Carteira conectada (MetaMask ou outra)
- ✅ Na rede BNB Smart Chain
- ✅ USDT (BEP-20) na sua carteira
- ✅ Algum BNB para taxas de gas (~$0,30)

## Obter USDT para Depósito

### Opção 1: Enviar de Exchange
1. Compre USDT na Binance, Coinbase etc.
2. Escolha rede **BEP-20 (BSC)** ao sacar
3. Envie para seu endereço de carteira
4. Aguarde confirmação (~3 minutos)

### Opção 2: Trocar na Carteira
1. Se você tem BNB ou outros tokens
2. Use o Swap integrado do MetaMask
3. Ou use PancakeSwap diretamente
4. Troque para USDT

## Depósito Passo a Passo

### Passo 1: Vá para Página de Depósito
Clique em "Depositar" no menu principal de imp.money

### Passo 2: Digite o Valor
- Mínimo: $10 USDT
- Máximo: Ilimitado
- Recomendado: Comece com um valor confortável

### Passo 3: Link de Indicação (Se houver)
Se alguém te convidou, o link de indicação já está aplicado.

### Passo 4: Aprovar Uso de USDT
1. Clique em "Aprovar"
2. Popup do MetaMask aparece
3. Confirme a transação
4. Aguarde confirmação

### Passo 5: Confirmar Depósito
1. Clique em "Depositar"
2. Popup do MetaMask para depósito
3. Verifique valor e gas
4. Confirme a transação

### Passo 6: Aguardar Confirmação
- Transação leva 3-15 segundos
- Você verá mensagem de confirmação
- Dashboard será atualizado

## Após o Depósito

### O que acontece agora?
- ✅ Depósito aparece no dashboard
- ✅ 0,7% ROI começa a acumular imediatamente
- ✅ Ciclo de 21 dias inicia
- ✅ Você pode resgatar lucros a qualquer momento

### Datas Importantes
- **ROI Começa**: Imediatamente
- **Primeiro Resgate**: Disponível após 24 horas
- **Capital Liberado**: Após 21 dias

## Perguntas Frequentes

**Por que duas transações?**
A primeira aprova o smart contract para usar USDT. A segunda executa o depósito real.

**E se a transação falhar?**
Verifique se tem BNB suficiente para gas. Tente novamente com limite de gas maior.

**Posso depositar mais?**
Sim! Cada depósito inicia seu próprio ciclo de 21 dias.
