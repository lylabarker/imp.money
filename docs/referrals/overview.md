# 21-Level Referral System

IMP Money features one of the most generous referral programs in DeFi - a 21-level deep commission structure paying **14% total** on all deposits.

## How It Works

When someone deposits using your referral link:

1. **You earn a commission** (paid instantly in USDT)
2. **They join your network** on Level 1
3. **Their referrals become your Level 2**, and so on
4. **You earn on 21 levels** of network depth
