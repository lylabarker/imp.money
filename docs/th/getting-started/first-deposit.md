# การฝากเงินครั้งแรก

คู่มือนี้จะนำคุณผ่านการฝาก USDT ครั้งแรกบน IMP Money

## ข้อกำหนดเบื้องต้น

ก่อนฝากเงิน:
- ✅ กระเป๋าเงินเชื่อมต่อแล้ว (MetaMask หรืออื่นๆ)
- ✅ อยู่บนเครือข่าย BNB Smart Chain
- ✅ มี USDT (BEP-20) ในกระเป๋าเงิน
- ✅ มี BNB สำหรับค่า Gas (~$0.30)

## รับ USDT สำหรับฝาก

### ตัวเลือก 1: ส่งจากตลาด
1. ซื้อ USDT จาก Binance, Coinbase ฯลฯ
2. เลือกเครือข่าย **BEP-20 (BSC)** เมื่อถอน
3. ส่งไปยังที่อยู่กระเป๋าเงินของคุณ
4. รอการยืนยัน (~3 นาที)

### ตัวเลือก 2: Swap ในกระเป๋าเงิน
1. ถ้าคุณมี BNB หรือ token อื่น
2. ใช้ Swap ในตัวของ MetaMask
3. หรือใช้ PancakeSwap โดยตรง
4. Swap เป็น USDT

## การฝากเงินทีละขั้นตอน

### ขั้นตอนที่ 1: ไปที่หน้าฝากเงิน
คลิก "Deposit" ในเมนูหลักของ imp.money

### ขั้นตอนที่ 2: ใส่จำนวนเงิน
- ขั้นต่ำ: $10 USDT
- สูงสุด: ไม่จำกัด
- แนะนำ: เริ่มต้นด้วยจำนวนที่คุณสบายใจ

### ขั้นตอนที่ 3: ลิงก์แนะนำ (ถ้ามี)
ถ้ามีคนเชิญคุณ ลิงก์แนะนำของพวกเขาถูกใช้แล้ว

### ขั้นตอนที่ 4: อนุมัติการใช้ USDT
1. คลิก "Approve"
2. MetaMask popup ปรากฏ
3. ยืนยันธุรกรรม
4. รอการยืนยัน

### ขั้นตอนที่ 5: ยืนยันการฝากเงิน
1. คลิก "Deposit"
2. MetaMask popup สำหรับฝากเงิน
3. ตรวจสอบจำนวนและ Gas
4. ยืนยันธุรกรรม

### ขั้นตอนที่ 6: รอการยืนยัน
- ธุรกรรมใช้เวลา 3-15 วินาที
- คุณจะเห็นข้อความยืนยัน
- แดชบอร์ดจะอัปเดต
