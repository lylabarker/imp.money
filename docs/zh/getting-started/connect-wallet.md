# 连接钱包

要使用IMP Money，您需要一个与BNB Smart Chain兼容的Web3钱包。

## 支持的钱包

### 🦊 MetaMask（推荐）
- Chrome、Firefox、Brave浏览器扩展
- iOS和Android移动应用
- 使用最广泛，文档最完善

### 💙 Trust Wallet
- 移动优先钱包
- 内置DApp浏览器
- 在亚洲流行

### 🔵 Coinbase Wallet
- 与Coinbase交易所分开
- 适合初学者的界面
- 浏览器扩展和移动版

### 🔗 WalletConnect
- 连接300+钱包
- 二维码连接
- 通用标准

## 设置MetaMask

### 步骤1：安装MetaMask
1. 访问 [metamask.io](https://metamask.io)
2. 点击"下载"
3. 选择您的浏览器或移动设备
4. 安装扩展

### 步骤2：创建钱包
1. 点击"创建新钱包"
2. 创建安全密码
3. **写下您的12个助记词**
4. 永远不要数字化存储
5. 确认您的助记词

### 步骤3：添加BNB Smart Chain
1. 点击网络下拉菜单（顶部）
2. 点击"添加网络"
3. 输入：
   - 网络名称：`BNB Smart Chain`
   - RPC URL：`https://bsc-dataseed.binance.org/`
   - Chain ID：`56`
   - 符号：`BNB`
   - 浏览器：`https://bscscan.com`
4. 保存

## 连接到IMP Money

### 步骤1：访问应用
前往 [imp.money](https://imp.money)

### 步骤2：点击"连接钱包"
在右上角找到按钮

### 步骤3：选择您的钱包
选择MetaMask（或您喜欢的钱包）

### 步骤4：批准连接
1. MetaMask弹窗出现
2. 验证网站（imp.money）
3. 点击"连接"
4. 确保您在BNB Smart Chain上

### 步骤5：验证
您的钱包地址现在应该显示在右上角

## 安全提示

⚠️ **保护您的助记词**
- 永远不要分享
- 不要数字化存储
- IMP永远不会要求它

⚠️ **验证URL**
- 始终直接访问imp.money
- 收藏官方网站
- 提防钓鱼网站

⚠️ **从小额开始**
- 先用小额测试
- 确保一切正常工作
