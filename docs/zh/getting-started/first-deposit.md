# 首次存款

本指南将引导您完成在IMP Money上的首次USDT存款。

## 前提条件

存款前请确保：
- ✅ 钱包已连接（MetaMask或其他）
- ✅ 已切换到BNB Smart Chain网络
- ✅ 钱包中有USDT（BEP-20）
- ✅ 有少量BNB用于Gas费（约$0.30）

## 获取存款用的USDT

### 选项1：从交易所发送
1. 在Binance、Coinbase等交易所购买USDT
2. 提现时选择**BEP-20（BSC）**网络
3. 发送到您的钱包地址
4. 等待确认（约3分钟）

### 选项2：在钱包内兑换
1. 如果您有BNB或其他代币
2. 使用MetaMask内置Swap
3. 或直接使用PancakeSwap
4. 兑换成USDT

## 分步存款指南

### 步骤1：进入存款页面
在imp.money主菜单点击"存款"

### 步骤2：输入金额
- 最低：$10 USDT
- 最高：无限制
- 建议：从您感到舒适的金额开始

### 步骤3：推荐链接（如有）
如果有人邀请您，他们的推荐链接已经应用。

### 步骤4：授权USDT使用
1. 点击"授权"
2. MetaMask弹窗出现
3. 确认交易
4. 等待确认

### 步骤5：确认存款
1. 点击"存款"
2. MetaMask存款弹窗
3. 检查金额和Gas
4. 确认交易

### 步骤6：等待确认
- 交易通常需要3-15秒
- 您将看到确认消息
- 仪表板将更新

## 存款后

### 接下来会发生什么？
- ✅ 存款显示在仪表板上
- ✅ 0.7% ROI立即开始累积
- ✅ 21天周期开始
- ✅ 您可以随时领取收益

### 重要日期
- **ROI开始**：立即
- **首次领取**：24小时后可用
- **本金解锁**：21天后

## 常见问题

**为什么需要两次交易？**
第一次授权智能合约使用USDT。第二次执行实际存款。

**如果交易失败怎么办？**
检查是否有足够的BNB用于Gas。尝试使用更高的Gas限制重试。

**我可以存更多吗？**
可以！每次存款都会开始自己的21天周期。
