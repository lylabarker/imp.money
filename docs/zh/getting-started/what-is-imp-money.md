# 什么是 IMP Money？

IMP Money（互联网货币协议）是BNB智能链上的去中心化DeFi协议，通过PancakeSwap V3的自动化流动性提供来产生收益。

## 问题

传统的DeFi收益耕作很复杂：
- 需要理解流动性池
- 手动管理仓位
- 重新平衡的高gas费用
- 无常损失风险
- 普通用户的技术障碍

## 解决方案

IMP Money简化一切：
- **一键存款** - 只需存入USDT
- **自动管理** - 协议处理流动性
- **稳定币重点** - 最小无常损失
- **每日回报** - 在您的存款上赚取0.7%日回报
- **网络奖励** - 建立团队以倍增收益

## 运作方式

```
您的USDT → IMP金库 → PancakeSwap V3 CLMM → 交易费 → 0.7%日收益
```

1. **您存入USDT** 到IMP智能合约
2. **协议部署** 到集中流动性池（USDT/USDC）
3. **交易费** 从交换量产生
4. **您赚取0.7%** 基于存入金额的日收益
5. **21天后** 您可以提取本金
