# ROI如何运作

IMP Money通过自动化流动性提供产生收益。以下是技术分解。

## 收益来源

您的回报来自**PancakeSwap V3交易费用**，而非新存款。

```
交易量 → LP费用 → IMP金库 → 您的每日ROI
```

### PancakeSwap V3 CLMM

IMP使用**集中流动性做市(CLMM)**：

- 资金部署在USDT/USDC交易对中
- 窄价格范围（稳定币交易接近1:1）
- 比传统AMM更高的资本效率
- 从范围内的每次交换中获取交易费

## 为什么是每日0.7%？

### 数学计算

- USDT/USDC池产生约50-100% APY费用
- 集中仓位可达100-500% APY
- 0.7%日收益 = 255% APY（复利）
- 扣除协议费用后，0.7%是可持续的

### 保守缓冲

协议保持储备以确保低交易量期间的提款。您的0.7%被设计为无论市场条件如何都保持一致。

## ROI计算

您的收益简单计算：

```
每日收益 = 存款金额 × 0.7%
```

### 示例

| 存款 | 每日 | 每周 | 每月 |
|------|------|------|------|
| $100 | $0.70 | $4.90 | $21.00 |
| $1,000 | $7.00 | $49.00 | $210.00 |
| $10,000 | $70.00 | $490.00 | $2,100.00 |

## ROI何时开始？

存款交易确认后立即开始赚取：

- 区块N：您存入$1,000
- 区块N+1：开始赚取
- 24小时后：约$7.00可领取

## 复利效果

如果每21天周期重新投资收益：

| 起始 | 6个月后 | 1年后 |
|------|---------|-------|
| $1,000 | $2,653 | $9,450 |
| $5,000 | $13,265 | $47,250 |
| $10,000 | $26,530 | $94,500 |

*假设完全再投资，不提取*
